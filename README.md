## What is Accio Work?

**Accio Work** is a desktop AI agent workspace that goes beyond chat — it *executes* real e-commerce workflows: sourcing suppliers, analyzing markets, launching Shopify stores, automating operations, and more.

> *"Not just a chatbox. A 24/7, self-evolving, proactive AI agent team that runs real business."*

## 🔌 Skills Ecosystem

This GitHub org hosts **open-source agent skills** — modular plugins that give AI agents domain-specific superpowers for e-commerce:

| Skill | What it does |
|-------|-------------|
| 🔍 **Product Selection** | Trend analysis, winning product discovery, category research |
| 🏭 **Supplier Sourcing** | Search 1.5M+ verified suppliers, compare quotes, auto-draft RFQs |
| 📊 **Market Insight** | Multi-source demand signals, competitor analysis, VOC research |
| 🛒 **Shopify Dropshipping** | End-to-end store launch: theme → listings → checkout → go-live |
| 📝 **Product Descriptions** | SEO-optimized listing copy for Amazon, Shopify, eBay, Etsy |
| ⭐ **Review Summarizer** | Scrape & analyze reviews from Amazon, Google, Yelp, TripAdvisor |
| 💰 **Tariff Calculator** | HS code classification, duty rates, landed cost estimation |
| 📈 **SEO Optimizer** | Page-level SEO audits and content optimization |
| 📣 **E-Commerce Marketing** | 19 specialized strategies: social media, ads, content, conversion |

> 🧩 Skills follow the **OpenClaw** & **MCP** standards — compatible with any agentic framework.

## 🚀 Quick Start

1. **Download** [Accio Work](https://www.accio.com/work) (macOS / Windows)
2. **Install a skill** from this org or from [ClawHub](https://clawhub.com)
3. **Tell your agent** what to do — in plain language
4. **Watch it execute**: browse the web, call APIs, write files, send emails — hands-free
